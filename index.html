Añadir vía repositorio

  1. En Kodi: Ajustes → Explorador de archivos → Añadir fuente.
  2. En <Ninguno> introducir esta dirección exacta:

    https://espatv.github.io/topzilla/
    Debe terminar en /topzilla/. La URL que acaba en .com devuelve la carpeta vacía.
  3. Asignar el nombre topzilla y aceptar.
  4. Volver al menú principal de Kodi. En Ajustes → Add-ons → caja abierta (arriba a la izquierda) → Instalar desde un archivo zip.
    (Si salta el aviso de seguridad: Ajustes → Sistema → Add-ons → activar "Orígenes desconocidos").
  5. Seleccionar la fuente topzilla → repository.topzilla-1.0.0.zip.
  6. Cuando aparezca "Add-on instalado", ir a Instalar desde repositorio → TopZilla Repository → Add-ons de vídeo → TopZilla → Instalar.

Vía ZIP

Misma instalación pero descargando el zip del repositorio directamente, sin configurar fuente:

  1. Descargar repository.topzilla-1.0.0.zip con el enlace de arriba.
  2. En Kodi: Ajustes → Add-ons → caja abierta → Instalar desde un archivo zip → seleccionar el archivo descargado.
  3. Continuar como en el paso 6 de arriba: Instalar desde repositorio → TopZilla Repository → Add-ons de vídeo → TopZilla → Instalar.
Si en lugar del repo descargas e instalas el addon directo (plugin.video.topzilla-X.X.X.zip), TopZilla funciona pero no recibirás actualizaciones automáticas al no tener el repositorio.
